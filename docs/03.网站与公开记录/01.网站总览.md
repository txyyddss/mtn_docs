`public.mcmtn.net` 是 MTNetwork 的公开记录站点。它不是单纯的宣传首页，而是把服务器运行状态、玩家目录、排行榜、在线历史与世界截图组织成一个持续更新的公共入口。

## 主站当前公开的主要区域

- 首页：服务器定位、实时状态、连接说明、公开叙事入口
- Players：玩家目录与单个玩家档案
- Core Members：核心成员介绍
- Gallery：世界截图与视觉档案

## 为什么文档站还需要说明主站

主站展示的是“结果”，文档站解释的是“这些结果该怎么理解”。例如：

- 为什么玩家页会有部分统计字段
- 为什么排行榜不应该被当成唯一评价标准
- 在线热力图展示的是什么采样结果
- 画廊更接近公共世界档案，而不是宣传素材堆积

## 推荐阅读路径

如果你第一次接触这些公开页面，建议按这个顺序理解：

1. [玩家档案](/public-records/player-profiles)
2. [排行榜](/public-records/leaderboards)
3. [在线状态与热力图](/public-records/live-status)
4. [画廊](/public-records/gallery)

## 使用原则

- 把网站当作公开记录，不是全部真相的替代品
- 把趋势数据当作趋势，不是绝对判断
- 把公开页和游戏内长期历史放在一起理解
