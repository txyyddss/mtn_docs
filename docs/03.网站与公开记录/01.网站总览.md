这篇网站总览解释 `public.mcmtn.net` 公开页面的组成、阅读入口以及文档站与主站之间的分工。

## 主站负责什么
主站负责展示“现在公开可见的内容是什么”，通常包括：

- 实时状态或当前入口
- 玩家目录与玩家档案
- 排行榜
- 在线状态与热力图
- 画廊或截图记录

如果你想知道当前公开页面有哪些、入口在哪里、数据有没有刷新，优先去主站看。

## 文档站负责什么
文档站不直接替代主站，而是负责解释：

- 这些页面为什么存在
- 每一类公开记录适合回答什么问题
- 哪些信息可以读，哪些结论不要过度外推
- 当页面与实际体验不一致时应如何理解

## 公开记录页面的组成
| 页面 | 适合看什么 | 不适合拿来做什么 |
| :--- | :--- | :--- |
| [玩家档案](/public-records/player-profiles) | 理解单个玩家公开资料页的含义 | 把档案页当作现实身份或完整行为画像 |
| [排行榜](/public-records/leaderboards) | 查看公开指标的排序结果 | 用单一榜单定义玩家全部价值 |
| [在线状态与热力图](/public-records/live-status) | 了解实时在线与周期性活跃趋势 | 精确追踪某人每一分钟做了什么 |
| [画廊](/public-records/gallery) | 回看世界截图与阶段性记录 | 把画廊当成完整世界备份 |

## 阅读这些页面时的边界
- 公开记录的存在是为了透明、可回看与可理解，不是为了无限度放大任何个人细节。
- 主站上的部分内容具有实时性，文档里的说明不会跟着每次页面微调同步改写。
- 如果你需要解释一页到底是什么意思，先回到对应文档页读“怎样理解”，再回主站看实时结果。
