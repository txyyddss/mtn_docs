## 为什么做公开记录

MTNetwork 的主站（[public.mcmtn.net](https://public.mcmtn.net/)）不只是一个连接信息展示页。它是服务器"长期生存 + 公开历史"理念的延伸——把玩家活动、世界变化和社区协作转化为可被后来者查阅的公共记录。

公开记录的核心原则：
- **透明**：服务器的运行状态、玩家活跃度和世界变化对所有人可见。
- **持久**：记录不会随存档重置或玩家离开而消失。
- **可追溯**：社区的发展脉络可以被任何人回看。

## 主站包含的主要内容

| 模块 | 说明 |
| :--- | :--- |
| 服务器状态 | 实时在线人数、服务器版本与连接地址 |
| 玩家列表 | 当前在线玩家与历史活跃记录 |
| 玩家档案 | 每位玩家的统计数据、活动轨迹与排行 |
| 排行榜 | 多维度玩家排名，包含 mcMMO 技能等类别 |
| 画廊 | 社区截图与世界记录 |
| 连接信息 | JE / BE 地址、端口与备用入口 |

## 数据来源与更新

- 所有数据由服务端自动采集并推送到主站，无需玩家手动上传。
- 更新频率因模块而异，状态信息接近实时，统计数据可能有短暂延迟。
- 如果发现数据异常，按 [反馈问题](/player-guide/report-issue) 流程提交。
