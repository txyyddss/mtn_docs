## 档案包含什么

每位在 MTNetwork 游玩过的玩家都会在主站生成一份公开档案，包含：

- **基本信息**：游戏内名称、UUID、首次加入时间
- **活跃数据**：在线时长、最后上线时间
- **统计数据**：挖掘、击杀、死亡、行走距离等原版统计项
- **技能数据**：mcMMO 各项技能等级与经验

## 排行榜

主站提供多维度排行榜，主要类别包括：

| 类别 | 说明 |
| :--- | :--- |
| 在线时长 | 累计游玩时间排名 |
| mcMMO 技能 | 各技能（挖掘、修理、战斗等）单项排名 |
| 原版统计 | 基于 Minecraft 原版统计数据的排名 |

排行榜数据定期刷新，可能存在短暂延迟。

## 隐私说明

- 公开档案中 **不会** 包含 IP 地址、设备信息或现实身份。
- 如果你希望降低在主站的可见度，可以减少可识别信息的暴露（参考 [社区与安全](/player-guide/community-safety) 中的隐私建议）。
- 公开记录是服务器运营透明度的一部分，无法对个别玩家完全隐藏基础统计。
