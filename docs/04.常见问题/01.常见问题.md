这篇常见问题汇总面向新玩家与访客最常见的一组高频疑问，适合在正式排错前先快速浏览。

## 加入前常问
| 问题 | 简短回答 |
| :--- | :--- |
| 这是商业服吗？ | 不是。MTNetwork 以非营利、长期生存和公开记录为核心。 |
| 文档里为什么没有实时 IP？ | 因为实时连接参数可能变化，这类信息统一以主站为准。 |
| Java 和 Bedrock 是两台服务器吗？ | 不是，是同一世界的不同接入方式。 |
| 需要先读哪些文档？ | 建议先看 [服务器简介](/getting-started/server-intro)、[加入方式](/getting-started/join) 和 [基础规则](/getting-started/rules)。 |

## 游玩与规则常问
| 问题 | 简短回答 |
| :--- | :--- |
| 这里更适合什么玩家？ | 更适合愿意长期建设、理解公共空间和公开记录的人。 |
| 可以只把它当资源服吗？ | 不建议。这里强调长期世界与公共历史，而不是短期索取。 |
| 可以使用客户端模组吗？ | 可以，但请先看 [模组兼容性](/player-guide/mod-compatibility) 中的推荐和禁止范围。 |
| 命令太多记不住怎么办？ | 先看 [服务器指令](/player-guide/commands)，再以服内帮助和 GUI 为准。 |

## 主站与公开记录常问
| 问题 | 简短回答 |
| :--- | :--- |
| 主站和文档站有什么区别？ | 文档站解释规则与含义，主站展示实时状态与公开页面。 |
| 排行榜能代表玩家全部价值吗？ | 不能。排行榜只是公开指标的排序结果。 |
| 热力图能精确追踪个人吗？ | 不应这样理解。它更适合看周期性活跃趋势。 |
| 画廊是不是完整世界备份？ | 不是。它是公开世界档案的一部分。 |

## 遇到问题先看哪里
- 连接失败、版本不符、客户端异常：去 [连接与排错](/faq/troubleshooting)。
- 规则边界不清楚：回看 [基础规则](/getting-started/rules) 和 [社区与安全](/player-guide/community-safety)。
- 需要正式反馈：看 [反馈问题](/player-guide/report-issue)。
