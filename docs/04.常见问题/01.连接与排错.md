## 连接前确认

在排查之前，先确认以下基础条件：

1. **已加入 QQ 群**（1064494318）并完成账户绑定。
2. **已查看主站**（[public.mcmtn.net](https://public.mcmtn.net/)）获取最新连接地址和端口。
3. **客户端版本**与服务器要求一致。

::: warning 端口可能变化
服务器采用家庭网络接入，IPv4 端口可能发生变化。始终以主站显示的实时参数为准。
:::

## 逐步排查

### 第一步：检查网络连通性
- 确认你的设备可以正常上网。
- 如果使用 IPv6，确认你的网络环境支持 IPv6 连接。
- 尝试 `ping` 主站地址，确认网络可达。

### 第二步：确认连接参数
- 打开主站，核对当前显示的 IP 地址和端口号。
- 如果主站显示有 SRV 记录或备用入口，优先尝试。
- 检查是否输入了多余的空格或拼写错误。

### 第三步：检查客户端
- 确认游戏版本号与服务器支持版本匹配。
- 如果使用模组客户端，尝试使用纯净客户端连接以排除模组冲突。
- BE 玩家确认已允许服务器资源包下载。

### 第四步：检查账号状态
- 确认你已在 QQ 群内完成绑定流程。
- 确认你的账号未被临时或永久封禁。
- 如果刚刚绑定，可能需要稍等片刻才能生效。

### 第五步：仍然无法连接
如果以上步骤都无法解决，按 [反馈问题](/player-guide/report-issue) 流程提交，并附上：
- 你的客户端类型和版本
- 连接时的错误信息截图
- 你使用的连接地址和端口
- 网络环境说明（IPv4 / IPv6、是否使用代理）

## 常见连接错误

| 错误信息 | 可能原因 | 建议操作 |
| :--- | :--- | :--- |
| Connection timed out | 地址或端口错误、网络不可达 | 核对主站连接参数 |
| Disconnected | 版本不匹配或未完成绑定 | 检查版本和绑定状态 |
| Server is full | 服务器已满 | 稍后重试 |
| Authentication failed | 账号验证失败 | 检查账号绑定和登录凭据 |
