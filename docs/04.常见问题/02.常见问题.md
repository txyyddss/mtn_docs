## 服务器相关

### 服务器支持哪些版本？
支持 Java Edition 和 Bedrock Edition 双版本互通。具体支持的版本号以主站显示为准。

### 服务器是正版验证还是离线模式？
请以 QQ 群公告中的说明为准。无论哪种模式，都需要在 QQ 群内完成账户绑定才能加入。

### 服务器会重置存档吗？
MTNetwork 以长期生存为核心理念，不会进行周期性存档重置。详见 [生存理念](/player-guide/survival-philosophy)。

### 服务器有哪些插件？
服务器安装了包括 mcMMO 技能系统、每日任务、领地保护、传送系统等多种插件。常用指令汇总见 [服务器指令](/player-guide/commands)。

## 账号与绑定

### 怎么加入服务器？
加入 QQ 群（1064494318）→ 按群公告绑定账号 → 在主站查看连接地址 → 打开游戏连接。详见 [加入方式](/getting-started/join)。

### Java 和 Bedrock 账号可以绑定吗？
可以。使用 `/linkplayer` 系列指令完成跨版本账号绑定。详见 [Java 与 Bedrock 互通](/getting-started/crossplay)。

### 绑定后数据会共享吗？
绑定后，两个版本的账号将共享游戏数据。

## 游玩相关

### 可以用模组吗？
可以使用纯客户端的辅助类模组（如小地图、合成查阅）。禁止矿透、自动挖掘和作弊客户端。详见 [模组兼容性](/player-guide/mod-compatibility)。

### 怎么设置家？
使用 `/sethome <名称>` 创建个人家，使用 `/home <名称>` 传送。详见 [服务器指令](/player-guide/commands)。

### 怎么保护我的建筑？
使用金铲子圈出区域，然后使用 `/dom create <名称>` 创建领地。领地内其他玩家默认无法破坏和交互。

### PVP 是默认开启的吗？
PVP 通过 `/pvp` 指令手动切换，未经双方同意的 PVP 行为违反 [基础规则](/getting-started/rules)。

## 主站与公开记录

### 主站上的数据是实时的吗？
状态信息接近实时，统计和排行数据定期刷新，可能存在短暂延迟。

### 我的信息会被公开多少？
主站会公开游戏内名称、在线时长、统计数据等信息。不会公开 IP 地址、设备信息或现实身份。详见 [玩家档案](/public-records/player-profiles)。
