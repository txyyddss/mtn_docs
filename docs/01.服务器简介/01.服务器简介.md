## 服务器简史

MTN 服务器于 2024 年正式创立。最初的核心团队由三位创始成员组成：负责硬件运维的 **mc9957**、资深玩家 **wor114514**。在创立初期，MTN 仅作为一个仅限 3 人好友间游玩的私人基岩版（Bedrock Edition）服务器。

**转折点：**
2024 年 5 月，随着软件负责人 **txrog** 的加入，服务器迎来了重大变革。他提出了“全民化”与“架构转型”的理念，主导将原有的封闭基岩版服务器转换为 Java 版服务器，以接纳更多玩家。随后，被称为铁路大亨的 **HEHAWellgood** 也加入团队。经过多次大版本迭代与性能调优，服务器最终决定正式对外开放。

## 历程碑与版本记录

### 🟢 起步阶段 (2024)

**2024年 开服 | 版本 0.x**
> **核心架构：** 使用 BDS 1.20.4 版本
- 纯原版生存，未添加任何插件。
- **地理志：** 出生点位于一片樱花树林，此处后来演变为服务器最古老的城市。
- **连接方式：** 处于封闭阶段，所有玩家必须通过 Zerotier 虚拟局域网连接。

**2024年 5月 | 版本 0.x**
- **架构转型：** 由 txrog 主导，使用存档转换器将基岩版存档无损迁移至 Java 版。
- **服务端：** 更换为 Paper 核心，并引入 Geyser 插件实现基岩版兼容。

**2024年 6月 1日 | 版本 1.0**
- **功能补完：** 引入任务系统、签到功能、在线地图及假人系统。
- **玩法拓展：** 新增血月、怪物等级等挑战性玩法。

**2024年 6月 2日 | 版本 2.0**
- **性能优化：** 服务端更换为 Putterfish 以提升并发能力。
- **内容增加：** 添加 mcMMO 技能系统和 BetterStructures 结构数据包。

**2024年 6月 11日 | 版本 3.0**
- **视觉与体验：** 引入动态照明、InteractionVisualizer（交互可视化）、Bluemap 及 Vane 插件。
- 进一步优化基岩版玩家的交互体验。

**2024年 6月 29日 | 版本 3.5**
- **常规维护：** 修复已知 Bug，引入自定义皮肤系统及悬浮文字指令。

---

### 🟡 深度定制与迭代 (2024 - 2025)

**2024年 7月 1日 | 版本 4.0**
- **工具链升级：** 集成 BestTools、CoreProtect（方块查询回滚）、AnimatedScoreboard（动态计分板）及 VeinMiner（连锁采集）。

**2024年 7月 10日 | 版本 5.0**
> **重大调整：** 优化了生存哲学，移除了体验欠佳的怪物等级与血月系统。
- **环境升级：** 引入 Terralith 地形数据包，使用 Chunky 工具清理无效区块冗余。
- **核心变更：** 服务端迁移至 PurPur。

**2024年 7月 26日 | 版本 5.1**
- **安全性：** 部署反作弊插件，深度调优全服插件配置。

**2024年 8月 16日 | 版本 6.0**
- **机制改动：** 临时调整为“死亡掉落”，增加 `/pvp` 指令开关功能。
- **精简化：** 移除了冗余的签到系统，针对村民 AI 进行了性能优化。

**2024年 12月 15日 | 版本 7.0**
- **系统重构：** 引入 AuthMe 登录插件，基础功能插件组更换为 CMI。
- **新功能：** 载具系统、地图画艺术、季节系统。
- **策略回归：** 重新开启“死亡不掉落”。

---

### 🔵 现代化与自动化 (2025)

**2025年 4月 18日 | 版本 6.0 (重置版)**
> **工程化突破：** 实现了基于 GitHub Actions 的自动打包与 Release 发布流程。
- **社区化：** 确立使用 GitHub Issues 作为玩家意见反馈的官方渠道。
- **版本跟进：** 升级至 Minecraft 1.21.3。

**2025年 4月 - 5月 | 快速迭代 (v6.1 - v6.3)**
- **6.1:** 全面修复玩家反馈问题。
- **6.2:** 修改 Paper 配置，恢复红石玩家所需的“刷沙机”功能。
- **6.3:** 跟进更新至 Minecraft 1.21.4。

**2025年 5月 24日 | 版本 6.4**
- **极致优化：** 服务端由 PurPur 更换为性能更佳的 **Leaf**。
- **界面优化：** 为基岩版定制了专属 GUI，使用 TAB 插件替换了原有的计分板系统。

**2025年 6月 - 7月 | 生态扩展 (v7.0 - v7.1)**
- **网络升级：** 支持 IPv4 & IPv6 双栈直连。
- **文档化：** 开始使用 Gitbook 编写服务器指南。
- **社会化：** 引入货币系统及玩家称号系统，基础插件更换为 SunLight。

---

### 🔴 重生与重构 (2026 - 至今)

**2026年 4月 9日 | 版本 8.0**
> **核心理念转型：回归原版，极致性能。**
- **技术突破：** 软件负责人回归，将服务端彻底重构为 **Lopine (Folia 分支)**。这种多线程架构彻底终结了服务器“土豆端”的历史。
- **系统瘦身：** 删除了季节等繁琐插件，仅保留功能性核心。
- **版本跨越：** 在保留存档的基础上，版本跨越式升级至 **1.21.11**。
- **开放筹备：** 优化后的服务器开始具备承载大量外部玩家的能力。

**2026年 4月 22日 | 现阶段**
- **数字化建设：** 服务器官网及实时玩家数据显示系统开发完成。
- **文档迁移：** 为了更好的访问体验，在线文档由 Gitbook 正式迁移至 **Vitepress**。
