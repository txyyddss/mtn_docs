## 服务器理念
服务器建立之初的愿景是腐竹与几位舍友为了能在任何时间一起消遣而建立的，如今开放是希望将当初那份欢乐分享与跟多人。服务器期望进行长周目运营，而非伴随短期存档的重置而导致记忆的消失。服务器的初衷是希望搭建一个富有社交性与娱乐性，使人们收获真面情绪与乐趣的地方。为了使其更加有趣，服务器在以Java版生存为主基调上，增添了不少更实用，趣味的插件，使其兼俱建筑，生电，冒险,经济交易等更多主题。愿你在游玩时，也能收获你的那份欢乐。
服务器的体验尽量靠近原版，不会因改件使游戏过度偏离原版，以微调维持公平，并适度替代或增加新玩法。服务器有双版本无障碍互通【Je/be】，无论你喜欢哪一种版本都可以畅玩本服。

## 服务器运营
服务器实行非营利运营、由玩家社区共同维护，共同创造。（一定要投喂腐竹的话我们也会很高兴哦：），代表我们的大土豆感谢您～～）

## 发展历程概览
| 阶段 | 时间 | 重点变化 |
| :--- | :--- | :--- |
| 起步阶段 | 2024 | 从好友间的封闭基岩版世界起步，随后迁移到 Java 侧并逐步开放。 |
| 深度迭代 | 2024 - 2025 | 经过多轮服务端、玩法和性能调整，逐步形成“长期生存 + 公开记录”的方向。 |
| 现代化建设 | 2025 | 引入 GitHub Issues、自动化发布流程与更完整的公开协作方式。 |
| 重构阶段 | 2026 至今 | 回归原版优先与极致性能，主站和 VitePress 文档体系逐步成形。 |
