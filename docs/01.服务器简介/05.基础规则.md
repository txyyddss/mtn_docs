## 总则

MTNetwork 以长期生存为核心，以下规则同时适用于游戏内、QQ 群和其他关联社区渠道。违反规则可能导致警告、临时禁止或永久封禁，具体视情节严重程度和历史记录而定。

## 破坏与盗窃
- 不得破坏任何公共设施
- 不在领地内的容器视为公共设施，允许适当取用
- 您可善用领地功能包含您的私有财产 恕不处理回滚请求

### 作弊与外挂
- 禁止矿透、自动挖矿、连点器和综合作弊客户端。
- 禁止利用 BUG 大量复制物品或获得不正当利益。
- 服务器已部署自动化反作弊机制，详见 [模组兼容性](/player-guide/mod-compatibility)。

### 恶意行为
- 禁止刷屏、骚扰、人身攻击或持续引战。
- 禁止在公共区域大规模破坏地形或制造噪音装置。
- 禁止未经对方同意的 PVP 行为。

## 公共空间与环境
- 公共交通线路和枢纽区域保持通畅，不得占用或堵塞。
- 建造项目与他人项目之间保持合理间距。
- 临时建筑和脚手架应在完工后清理。

## 处罚机制
| 违规项目 | 情形 | 处理方式 |
| :--- | :--- | :--- |
| 作弊 | 使用作弊客户端或模组 | 由反作弊系统自动踢出，不会进行封禁 |
| 破坏 | 恶意破坏公共设施 | 封禁1个月 |
| 不当言论 | 在游戏内或群组内发布人身攻击或违反法律的言论 | 禁言并保存档案、提交至中国警方 |

## 申诉
如果你认为处罚存在误判，可以在QQ群组内提交申诉。申诉需提供具体时间、上下文和你认为误判的理由。
