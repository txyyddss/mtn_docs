## 总则

MTNetwork 以长期生存为核心，以下规则同时适用于游戏内、QQ 群和其他关联社区渠道。违反规则可能导致警告、临时禁止或永久封禁，具体视情节严重程度和历史记录而定。

## 禁止行为

### 破坏与盗窃
- 未经同意，不得破坏、拆除或改造他人建筑与设施。
- 不得擅自取走他人容器内的物品。
- 公共设施的改造需先在社区渠道沟通。

### 作弊与外挂
- 禁止矿透、自动挖矿、连点器和综合作弊客户端。
- 禁止利用 BUG 大量复制物品或获得不正当利益。
- 服务器已部署自动化反作弊机制，详见 [模组兼容性](/player-guide/mod-compatibility)。

### 恶意行为
- 禁止刷屏、骚扰、人身攻击或持续引战。
- 禁止在公共区域大规模破坏地形或制造噪音装置。
- 禁止未经对方同意的 PVP 行为（PVP 状态可通过 `/pvp` 开关）。

## 公共空间与环境
- 公共交通线路和枢纽区域保持通畅，不得占用或堵塞。
- 建造项目与他人项目之间保持合理间距。
- 临时建筑和脚手架应在完工后清理。

## 处罚机制
| 级别 | 情形 | 处理方式 |
| :--- | :--- | :--- |
| 提醒 | 初次轻微违规 | 口头提醒，记录备案 |
| 警告 | 重复轻微违规或单次中度违规 | 正式警告 + 记录 |
| 临时封禁 | 严重违规或累计警告 | 限时禁止进入服务器 |
| 永久封禁 | 恶意破坏、作弊或危害社区安全 | 永久封禁，不可申诉 |

## 申诉
如果你认为处罚存在误判，可以通过 [反馈问题](/player-guide/report-issue) 中的流程提交申诉。申诉需提供具体时间、上下文和你认为误判的理由。
