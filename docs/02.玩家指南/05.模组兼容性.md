## 基本原则
- 本服务器支持 **无模组纯净游玩**，你不需要安装任何模组也能加入。
- 推荐模组的目标是改善信息可读性、地图查看和建筑协作，而不是替代正常游玩。
- “客户端模组” 不等于 “默认允许一切增强工具”；影响公平和防护机制的工具仍然禁止。

::: tip 关于生电辅助类模组
虽然服务端不会为所有生电玩家工具提供专门兼容代码，但并不默认禁止纯客户端的阅读与规划辅助类模组。
:::

## 推荐与兼容模组
以下类型更适合作为公开玩家的常用配置：

| 模组名称 | 功能说明 | 推荐配置 / 备注 |
| :--- | :--- | :--- |
| **Distant Horizons** | 超远视距渲染 | 请在设置中 **开启** `Server Generation` 并 **关闭** `Enable Distant Generation`。最大视距限制为 **128** 区块。 |
| **REI (Roughly Enough Items)** | 合成查阅 | 强大的物品清单与合成表显示工具。 |
| **Jade 🔍** | 视线信息显示 | WTHIT 的高级替代品，实时显示准星所指方块/实体信息。 |
| **Syncmatica** | 投影共享 | 允许您与其他玩家实时共享 **Litematica** 的建筑投影。 |
| **AppleSkin** | 饮食信息可视化 | 在工具栏精确显示饱食度、饱和度及食物恢复效果。 |
| **BBOR** | 结构边界显示 | 生电玩家必备，用于可视化村庄、要塞等结构范围。 |
| **Xaero's Maps** | 游戏内地图 | 包含小地图 (Minimap) 与世界地图 (World Map) 系列。 |

## 高风险与禁止类型
服务器配有自动化防护机制。请不要把“客户端自定义”理解成可以挑战公平与安全边界。

::: danger 安全警告
任何明显以透视、自动化挖掘、异常交互频率或综合作弊为目标的工具，都不属于允许范围。
:::

| 违规类型 | 详细说明 | 防御机制状态 |
| :--- | :--- | :--- |
| **矿透 & 自动挖矿** | 任何形式的透视矿石或自动寻路挖掘。 | **已启用** 强力假矿模块，当开启矿透时，您将看到铺天盖地的矿石。 |
| **玩家透视 (X-Ray)** | 试图隔墙观察其他玩家位置。 | **已启用** 光线追踪算法，自动剔除视野外玩家实体包。 |
| **连点器** | 超高频率的交互或攻击行为。 | **已启用** 行为检测，异常频率的操作将被自动撤销。 |
| **作弊客户端** | 如彗星端等综合性作弊工具。 | **已启用** 实时动作监测，所有违规动作均会被服务器回滚。 |

## 安装前的自检
- 先确认模组是客户端侧工具，而不是会主动绕过服务端规则的作弊组件。
- 遇到冲突时，优先最小化复现：只保留必要模组再测试。
- 不确定某个模组是否踩线时，宁可先问再装。
