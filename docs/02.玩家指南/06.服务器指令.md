## 📱 基岩版 GUI 菜单

> 专为基岩版玩家设计的便捷菜单指令，输入后即可开启可视化交互界面。


`/warpgui`
:打开**传送点 GUI**，选择目标传送点进行传送

`/tpgui`
:打开**传送请求 GUI**，选择要传送的玩家。

`/homegui`
:打开**个人家列表 GUI**，快速传送回家。

`/phomegui`
:打开**公共家列表 GUI**，访问其他玩家的公共家。

`/msggui`
:打开**消息发送 GUI**，方便与其他玩家交流。

## 🏕️ 领地系统

::: tip 💡 圈地前置操作
在执行创建领地指令前，请务必先使用 **金铲子** 在地上圈出您想要的领地范围！
:::

`/dom create <领地名称>`
: 将已圈选的区域创建为正式领地。

`/dom set_tp <领地名称>`
: 更新或设置指定领地的默认传送落脚点。

`/dom tp <领地名称>`
: 瞬间传送到您的指定领地。

## 🔗 账户绑定 (Java 与基岩互通)

::: info 📝 绑定流程说明
绑定账户可让您的 Java 版和基岩版共享同一游戏进度与数据。
:::

`/linkplayer <对方账户名>`
: 📤 **发起绑定**：向目标账户发送绑定请求。

`/linkplayer <对方账户名> <验证码>`
: 📥 **接受绑定**：输入收到的验证码以确认绑定。

`/unlinkplayer`
: ✂️ **解除绑定**：在基岩版执行此指令可解除当前的账户绑定状态。

## 🤸 趣味动作

使用以下指令让你的角色做出各种生动的动作：

- **坐下**： `/sit`
- **躺下**： `/lay`
- **趴下**： `/bellyflop`
- **旋转**： `/spin`
- **爬行**： `/crawl`

## 🎯 每日任务

`/dq me`
: 📜 打开每日任务面板 GUI。

`/dq reroll <任务编号>`
: 🔄 刷新指定编号的每日任务（用于替换不想做的任务）。

## 👕 皮肤设置

::: tip ✅ 正版玩家特权
如果您拥有 **Minecraft 正版账号**，系统会自动同步您的皮肤，**无需进行任何额外设置**！
:::

对于非正版玩家或想要更换自定义皮肤的玩家，可以使用以下指令：

`/skin set <正版玩家名>`
: 使用指定正版玩家的皮肤外观。

`/skin url "<图片链接>"`
: 使用 **自定义图片链接** 设置皮肤（注意：链接前后需要保留双引号 `""`，且图片须为标准的皮肤格式）。

`/skin clear`
: 🧹 **清除当前皮肤**，恢复为默认外观。

## 🎲 杂项与综合功能

`/if create <名称> <图片链接> <宽> <高>`
: 🖼️ **创建地图画**：将网络图片直接生成为游戏内的地图画。

`/rewards claim`
: 🎁 **每日签到**：领取您的日常在线奖励。

`/pvp`
: ⚔️ **PVP 状态切换**：一键开启或关闭您的玩家对战状态。
