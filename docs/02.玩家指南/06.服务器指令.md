## 使用前先知道
- 服内帮助、菜单 GUI 与当前版本公告永远比静态文档更实时。
- 这页只保留最常用、最适合公开玩家快速检索的一组命令。
- 如果你发现某条命令在当前版本不可用，先以服内提示为准，再按 [反馈问题](/player-guide/report-issue) 提交修正。

## 菜单与传送

| 指令 | 用途 | 备注 |
| :--- | :--- | :--- |
| `/warpgui` | 打开传送点 GUI | 适合快速浏览管理员维护的公共传送点。 |
| `/tpgui` | 打开传送请求 GUI | 图形化选择要发起传送的对象。 |
| `/homegui` | 打开个人家列表 GUI | 快速返回个人家的入口。 |
| `/phomegui` | 打开公共家列表 GUI | 查看其他玩家公开的家。 |
| `/bgui open main` | 打开主指令菜单 | 基岩版玩家最常用的综合入口。 |
| `/home <名称>` | 传送到个人家 | 需要你已经创建对应家。 |
| `/sethome <名称>` | 以当前位置创建家 | 用于设置个人常用落点。 |
| `/homelist` | 查看个人家列表 | 适合先确认名称再传送。 |
| `/delhome <名称>` | 删除个人家 | 清理不再使用的落点。 |
| `/edithome <名称> privacy [public\|private]` | 修改家的公开状态 | 决定该家是否出现在公共家列表。 |
| `/phome <名称>` | 传送到公开家 | 仅对公开家生效。 |
| `/phomelist` | 查看公开家列表 | 访问他人公开共享的落点。 |
| `/warplist` | 查看管理员传送点 | 常与 `/warp <名称>` 配合使用。 |
| `/warp <名称>` | 传送到管理员传送点 | 以公共入口为主。 |
| `/rtp` | 随机传送 | 通常会消耗一定资源或金钱。 |
| `/tpaccept` | 接受传送请求 | 处理别人发来的传送请求。 |
| `/tpdecline` | 拒绝传送请求 | 不接受时使用。 |
| `/tpa <用户名>` | 请求传送到目标玩家 | 常规玩家传送请求。 |
| `/tpahere <用户名>` | 请求目标玩家传送到你这里 | 适合邀请对方到当前地点。 |

::: tip 💡 圈地前置操作
在执行创建领地指令前，请先使用 **金铲子** 圈出希望保护的区域。
:::

## 领地与公共空间

| 指令 | 用途 | 备注 |
| :--- | :--- | :--- |
| `/dom create <领地名称>` | 创建领地 | 基于你已经圈出的区域创建正式领地。 |
| `/dom set_tp <领地名称>` | 设置领地传送点 | 更新默认落脚位置。 |
| `/dom tp <领地名称>` | 传送到领地 | 快速返回自己的领地。 |
| `/dom info` | 查看当前位置领地信息 | 判断当前区域归属和状态。 |

## 社交与绑定

| 指令 | 用途 | 备注 |
| :--- | :--- | :--- |
| `/msggui` | 打开消息 GUI | 便于快速选择消息对象。 |
| `/anon <消息内容>` | 发送匿名消息 | 用于降低直接暴露身份。 |
| `/ball <提问内容>` | 触发“答案之书” | 趣味向随机回答。 |
| `/linkplayer <对方账户名>` | 发起绑定请求 | 用于 Java 与 Bedrock 账户绑定。 |
| `/linkplayer <对方账户名> <验证码>` | 接受绑定请求 | 输入验证码确认绑定。 |
| `/unlinkplayer` | 解除绑定 | 基岩版常用的解绑入口。 |

## 外观、任务与趣味功能

| 指令 | 用途 | 备注 |
| :--- | :--- | :--- |
| `/dq me` | 打开每日任务面板 | 查看当前每日任务。 |
| `/dq reroll <任务编号>` | 重掷每日任务 | 替换不想做的指定任务。 |
| `/skin set <正版玩家名>` | 套用指定正版玩家皮肤 | 常用于非正版玩家或外观切换。 |
| `/skin url "<图片链接>"` | 使用图片链接设置皮肤 | 链接前后需要保留双引号。 |
| `/skin clear` | 清除当前皮肤设置 | 恢复默认外观。 |
| `/sit` / `/lay` / `/bellyflop` / `/spin` / `/crawl` | 趣味动作 | 让角色做出不同动作。 |
| `/if create <名称> <图片链接> <宽> <高>` | 创建地图画 | 把网络图片生成到游戏内地图画。 |
| `/rewards claim` | 领取在线奖励 | 当前版本的奖励领取入口。 |
| `/pvp` | 切换 PVP 状态 | 一键开启或关闭玩家对战状态。 |

::: tip ⚔️ mcMMO 技能系统
本服务器已实装 `mcMMO` 技能插件。详细技能说明请查阅 [mcMMO 官方 Wiki](https://wiki.mcmmo.org/)。
:::
